C-Truck Documentation

Adding Vehicle Data

To add a new vehicle to the system, follow these steps:

  1. Go to the "Register Vehicle" page.
  2. Fill out the registration form with the required vehicle information.
  3. Click the "Register Vehicle" button to submit the form.
  4. You'll be redirected to the "Track" page with a success message if the registration is successful.

Deleting Vehicle Data

To delete vehicle data, you have two options:

Option 1: Using the Django Admin Interface

  1. Log in to the Django admin interface.
  2. Go to the "Vehicles" section.
  3. Select the vehicle(s) you want to delete.
  4. Click the "Delete selected vehicles" button.

Option 2: Using the "Manage Vehicles" Page

  1. Go to the "Manage Vehicles" page.
  2. Find the vehicle you want to delete in the table.
  3. Click the "Delete" button next to the vehicle.
  4. Confirm the deletion when prompted.

Other Important Information

Here are some other things to keep in mind:

  • Make sure you're logged in to perform any data management operations.
  • Always back up your data before performing any deletions.
  • If you encounter any issues, check the Django server logs for error messages.
